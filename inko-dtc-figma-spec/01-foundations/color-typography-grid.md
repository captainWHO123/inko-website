## Grid\
\
### Desktop\
- Canvas width: 1920px\
- Content container: 1280px\
- Main grid: 12 columns optional\
- Three-card grid:\
  Columns: 3\
  Column width: full width\
  Gap: 0px\
\
### Tablet\
- Breakpoint: below 960px\
- Content container: calc(100vw - 48px)\
- Three-card grid becomes 1 column unless specified.\
- Two-column hero becomes 1 column.\
\
### Mobile\
- Breakpoint: below 750px\
- Content container: calc(100vw - 32px)\
- Cards stack vertically.\
- Header nav collapses into menu.\
- Forms stack input above button.\
- Media containers use aspect ratio instead of fixed height.\
\
